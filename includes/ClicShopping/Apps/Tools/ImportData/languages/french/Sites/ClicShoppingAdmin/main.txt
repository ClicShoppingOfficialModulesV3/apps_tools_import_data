text_help_description = <strong>Note à lire avant une migration</strong>
<blockquote>
Pour que la migrations des données puissent se faire correctement,<br />
<li>Vous ne devez pas avoir de données dans votre nouvelle base de données sauf celles importées d'origine</li>
<li>Veuillez toujours faire un test de migration avec une base de données test afin de bien vérifier le bon fonctionnement de la migration</li>
<li>Les données importées se font au format brut, chaque table est lue et importée dans la base de données de ClicShopping.</li>
<li>Si une erreur venait à se produire pendant la phase de la migration, veuillez l'identifier et faire les corrections.</li>
<li>ClicShopping a 2 langues par défaut, les données seront répliquées dans les 2 langues de ClicShopping. Il vous faudra traduire par la suite la langue importée.
<li>Nous déclinons toutes reponsabilités en cas d'erreur d'importation</li>
</blockquote>
